# HDD Scheduling Algorithms Library
General purpose library for simulating various disk scheduling algorithms

## Currently supported algorithms:
- FIFO
- FCFS
- SCAN
- CSCAN
- LOOK
- CLOOK

### Dependencies:
Usage of virtualenv is encouraged.
